Customer Behavior Analysis Project

Overview

This project demonstrates an end-to-end data analytics workflow focused on analyzing customer shopping behavior. It covers data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and building an interactive dashboard to generate meaningful business insights.

The objective is to understand customer purchasing patterns, evaluate the impact of discounts, and analyze product performance to support data-driven decision-making.

---

Dataset

The dataset contains structured customer transaction data, including:

- Customer demographics (e.g., gender, age group)
- Purchase amount and product categories
- Discount usage
- Ratings and shipping details

---

Tools and Technologies

- Python (Pandas, Matplotlib, Seaborn)
- SQL (MySQL / PostgreSQL / SQL Server)
- Power BI
- MS Word
- Gamma AI

---

Project Workflow

1. Data Loading and Cleaning

- Loaded dataset using Python (Pandas)
- Handled missing values and inconsistencies
- Removed duplicates and corrected data types

2. Exploratory Data Analysis (EDA)

- Analyzed customer purchase patterns
- Compared spending behavior across segments
- Visualized trends using charts and graphs

3. SQL Analysis

- Revenue distribution by gender
- Impact of discounts on spending
- Top-rated products
- Customer segmentation

4. Dashboard Development

- Built an interactive dashboard in Power BI
- Used filters and slicers for dynamic analysis
- Visualized key metrics such as revenue and customer segments

5. Reporting and Presentation

- Created a detailed project report (PDF)
- Developed a presentation using Gamma AI

Results and Insights

- Identified top-performing product categories

How to Run

1. Download or clone the repository
2. Open the ".ipynb" file in Jupyter Notebook
3. Run all cells to perform data analysis
4. Execute SQL queries in your database (MySQL / PostgreSQL / SQL Server)
5. Open the ".pbix" file in Power BI Desktop
6. Review the report and presentation for final insights
